# Thuật ngữ & Viết tắt (Glossary)

| Thuật ngữ | Ý nghĩa |
|---|---|
| **HLV** | Huấn luyện viên dinh dưỡng (nhà phân phối Herbalife) |
| **KH / KHTN** | Khách hàng / Khách hàng tiềm năng (lead) |
| **DISC** | Mô hình tính cách D / I / S / C (thẻ phụ, điều chỉnh tông giọng) |
| **Stage-of-Change** | Giai đoạn sẵn sàng thay đổi (precontemplation → maintenance) |
| **Persona** | Chân dung khách hàng (Aim + Pain + Stage + DISC) |
| **Tanita** | Cân phân tích thành phần cơ thể (cân nặng, mỡ, cơ, nước, mỡ nội tạng, tuổi sinh học, BMR) |
| **TMR / RMR / AMR** | Tổng / Nghỉ / Hoạt động — năng lượng tiêu hao (kcal/ngày) |
| **EX** | Năng lượng tiêu hao do vận động |
| **"Con số kỳ diệu"** | TMR = RMR + AMR + EX (tổng calo mục tiêu/ngày) |
| **Đồng hồ sinh học** | Widget 24h trên màn "Sức khỏe tổng thể" (KH), chia múi theo hoạt động |
| **Thân – Tâm – Trí** | 3 trụ cột lộ trình: Thể chất · Lối sống/cảm xúc · Kiến thức (tài chính/kinh doanh) |
| **NBA** | Next-Best-Action — gợi ý hành động kế tiếp (lớp AI nâng cao, sau MVP-3) |
| **DMO** | Daily Method of Operation — phương pháp vận hành hằng ngày |
| **GNV** | Giấy nhắc việc |
| **HOM** | Buổi sinh hoạt nhóm dinh dưỡng (Home Meeting) |
| **PSM** | Process State Model — La bàn quy trình 12 bước (xương sốh hệ thống) |
| **DSKHTN** | Danh sách khách hàng tiềm năng |
| **2/1** | Buổi gặp 2 người (HLV + khách tiềm năng) + 1 người dẫn (mentor) — bước chốt gặp |
| **Talking point** | 21 bài kiến thức dinh dưỡng cơ bản (chăm sóc KH) |
| **Feasibility Score** | Điểm khả thi bữa ăn (chỉ dùng nội bộ, **không in số ra UI** — dịch thành nhãn Dễ/Hơi khó/Khó) |
| **Persona-fit** | Lọc thực đơn theo chế độ ăn/ngân sách/sở thích (chỉ dùng nội bộ) |
| **Content-Attribution Matching** | Cơ chế khách tự gắn về HLV qua mã giới thiệu trên nội dung chia sẻ |
| **DXP** | Digital User Experience Platform |
